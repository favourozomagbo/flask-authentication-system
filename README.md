# Flask Authentication App 🔐

A secure authentication system built with Flask that allows users to create accounts, log in, log out, and access protected pages. The application uses SQLite for data storage, password hashing for security, session management for authentication, and password strength validation during registration.

## Features

* User Registration (Sign Up)
* User Login
* User Logout
* Protected Dashboard
* Session Management
* Password Hashing with Werkzeug
* SQLite Database Integration
* Flash Messages for User Feedback
* Password Strength Indicator
* Strong Password Validation
* Responsive User Interface
* Navigation Bar and Dashboard Statistics

## Technologies Used

* Flask

## Installation

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project folder:

```bash
cd Authentication-App
```

3. Install Flask:

```bash
pip install flask
```

4. Run the application:

```bash
python app.py
```

5. Open your browser and visit:

```text
http://127.0.0.1:5000
```
